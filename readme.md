## 开发工具和平台

 - 环境：win10
 - IDE：IDEA
 - 语言：Java

## 实现功能

 - 查看是否在线
 - 登陆
 - 互发消息
 - 传送txt文件

## 具体实现
数据的传送是使用UDP协议实现的，在线情况通过查询日志文件获取当前在线信息。
